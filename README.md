# Basic-Video-Example



## AR视频通话

#### 快速开始 🔜

本示例包含了 RTC 4.0 SDK 以及集成 Demo

Demo演示了以下功能

- 创建 RtcEngine 实例
- 加入频道
- 显示自己和对方的视频



#### 集成SDK🔜

你可以查看文档了解如何 [快速集成](https://docs.anyrtc.io/rtc-android/docs/quickstart)

完整文档地址 👉 [API参考](https://docs.anyrtc.io/)
